# Assignment 01

Follow the instructions provided in the course material.

Submissions must be made through a Pull Request.

## Rules

* Do not modify files outside your own submission folder.
* Do not push directly to `main`.
* Use a dedicated branch for your submission.
* Follow the Pull Request template when opening your submission.

## Submission

Open a Pull Request from your fork to the original repository.

The Pull Request itself is considered the submission and does not need to be merged.

Good luck.
